# ポモドーロタイマー 実装機能一覧

## 1. タイマー機能（中核機能）

- [ ] 25分のポモドーロセッション開始
- [ ] 5分の短い休憩タイマー
- [ ] 15分の長い休憩タイマー
- [ ] タイマーのカウントダウン表示（MM:SS形式）
- [ ] タイマーの開始/一時停止/再開機能
- [ ] タイマーのリセット機能
- [ ] セッション完了時の自動通知・遷移（休憩へ）
- [ ] 休憩完了時の自動遷移（次セッションへ）

## 2. UI・表示機能

- [ ] 円形プログレスバー表示（残り時間の進捗可視化）
- [ ] 現在のステータス表示（「作業中」「休憩中」など）
- [ ] 残り時間数値表示（25:00などの大きいフォント）
- [ ] 開始ボタン（デザイン）
- [ ] リセットボタン（デザイン）
- [ ] ボタン状態の管理（有効/無効の切り替え）
- [ ] 画面レイアウト・スタイリング

## 3. 進捗管理機能

- [ ] セッション完了履歴の記録
- [ ] 本日の完了セッション数の集計
- [ ] 本日の累計集中時間の計算
- [ ] 「今日の進捗」カードの表示
  - [ ] 完了数量表示
  - [ ] 累計集中時間表示

## 4. 設定機能

- [ ] ポモドーロセッション時間の設定（デフォルト：25分）
- [ ] 短い休憩時間の設定（デフォルト：5分）
- [ ] 長い休憩時間の設定（デフォルト：15分）
- [ ] 設定値の保存・読み込み

## 5. データベース機能

- [ ] SQLiteデータベースセットアップ
- [ ] セッション完了記録テーブル
- [ ] 設定値管理テーブル
- [ ] リポジトリパターンの実装（Session、Settings）

## 6. バックエンド API機能

- [ ] セッション完了エンドポイント（記録API）
- [ ] 本日の進捗取得エンドポイント
- [ ] 設定値取得エンドポイント
- [ ] 設定値更新エンドポイント

## 7. テスト関連

- [ ] ユニットテスト環境設定
- [ ] タイマーロジックのテスト
- [ ] 進捗計算のテスト
- [ ] API通信のテスト

## 8. その他

- [ ] 音声通知またはアラート機能（セッション完了時）
- [ ] ブラウザタイムアウト対策（タイマー精度維持）
- [ ] エラーハンドリング
- [ ] ロギング機能
